# my-tripmate-wepapp

Trip planning & voting web application

---

## Tech Stack
- Frontend: React + TypeScript
- Backend: Node.js / API
- State & UI: Custom components
- Tooling: Vite, Git, npm

---

## Project Structure
```
my-tripmate-wepapp
├─ frontend
├─ backend
```

---

## Frontend Scripts
```bash
npm run dev # vite
npm run build # vite build
npm run preview # vite preview
```

---

## Setup

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm start
```

---

## Main Features
- Trip voting system
- Member progress tracking
- Budget & place selection
- Summary & analytics
- Owner / member role separation
